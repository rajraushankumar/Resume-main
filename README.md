📄 Rajraushan Resume

This repository contains my professional resume built using LaTeX. It includes all necessary files required to generate a clean and modern resume in PDF format.

📁 Project Structure
Raj resume 2.tex – Main LaTeX source file
Raj resume 2.pdf – Final resume output
Other files (.aux, .log, .out, etc.) – Auto-generated during compilation
🚀 How to Run
Install LaTeX (TeX Live / MiKTeX)
Open .tex file in any editor (VS Code / Overleaf)

Compile using:

pdflatex "Raj resume 2.tex"
Your resume PDF will be generated
